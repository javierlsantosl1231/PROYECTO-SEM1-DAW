# Proyecto de Asignatura - Desarrollo Web - VERSIÓN COLORES VIVOS

Este repositorio contiene la estructura básica de una página web en HTML5 según los requerimientos de la actividad, mejorada con una paleta de colores vivos y estilos CSS internos.

## Contenido del repositorio
- `index.html`: Archivo principal con la estructura de encabezados (H1, H2, H3) y estilos CSS internos para colores vivos.

## Comandos para subir este proyecto a GitHub desde la terminal de VS Code:

1. Inicializar el repositorio Git local:
   git init

2. Añadir el archivo index.html al área de preparación:
   git add index.html

3. Confirmar los cambios con un mensaje descriptor:
   git commit -m "feat: mejorada la página con estilos CSS de colores vivos"

4. Renombrar la rama a main:
   git branch -M main

5. Vincular con tu repositorio remoto de GitHub (reemplaza TU_USUARIO y TU_REPOSITORIO):
   git remote add origin https://github.com/TU_USUARIO/TU_REPOSITORIO.git

6. Subir los archivos de forma definitiva:
   git push -u origin main
