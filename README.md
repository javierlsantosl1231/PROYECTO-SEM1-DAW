# Proyecto de Asignatura - Desarrollo de Aplicaciones Web 

Este repositorio contiene la estructura básica de una página web en HTML5 según los requerimientos de la actividad y con diseño original del autor.

## Contenido del repositorio
- `index.html`: Archivo principal con la estructura de encabezados (H1, H2, H3) y estilos CSS internos para diseños originales.

## Instrucciones para desarrollar la tarea:

Abra en Visual Studio Code el proyecto trabajado en la Semana 2.
Verifique que el archivo principal se llame exactamente index.html.
Complete la estructura básica de HTML5 utilizando DOCTYPE, html, head, meta charset, title y body.
Organice la página utilizando etiquetas semánticas: header, nav, main, section, article, aside y footer.
Dentro del header, coloque el título principal del proyecto utilizando h1.
Dentro del nav, agregue un menú con enlaces internos: Inicio, Quiénes Somos, Servicios o Productos y Contacto.
Cree una sección llamada Inicio con una breve presentación del proyecto.
Cree una sección llamada Quiénes Somos utilizando section, h2 y p. Esta sección debe contener una descripción de al menos 100 palabras sobre el proyecto, emprendimiento o sistema web.
Agregue una imagen relacionada con el proyecto dentro de la sección Quiénes Somos.
Cree una sección llamada Servicios o Productos utilizando una lista HTML con ul y li.
Cree una sección llamada Contacto que incluya correo electrónico, teléfono, dirección y enlaces a redes sociales mediante la etiqueta a.
Incorpore un video de YouTube relacionado con el proyecto o actividad económica mediante la etiqueta iframe.
Agregue información complementaria dentro de la etiqueta aside, como promociones, horarios de atención o información relevante.
Agregue un footer con el nombre del estudiante, asignatura y año.
Guarde los cambios, visualice la página en el navegador y suba el archivo actualizado al repositorio de GitHub.
